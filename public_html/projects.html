Ant Game

The AntGame Project was written in java as a coursework assignment for the software engineering university course.
The program simulates two ant colonies competing for food clusters on procedural maps.
The ants with the most food on the their ant hill at games end win.
The only form of player control in the game comes from designing the ants brains and the maps.

The Maze game was written in java as a coursework assignment for the further programming university course.
The game is a simple two dimensional maze with a controllable explorer.
Further functionality was included though the implementation of random, depth first search and traverse-all-cells functions to find the maze end and ensure the maze end was reachable respectively.
